    • Datos de la turbina eólica

    Aca se pueden ver los datos tomados de una turbina eólica. Estos serán analizados.

    Potencia activa

    Este dato refleja la potencia real que esta generando el sistema de energía eólica.

    Potencia teorica

    Este dato refleja la potencia que deberia generar el sistema, si se tomase como ideal.

    Velocidad del viento

    Este dato es sumamente importante para poder analizar la correlación que este tiene con la potencia obtenida.

    Dirección del viento

    Este dato presenta información necesaria para saber en que dirección poner las turbinas para mayor eficiencia.
